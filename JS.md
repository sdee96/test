JS
-----

JS(JavaScript)는 객체(Object)기반의 스크립트 언어이다  

HTML로는 웹의 구조를 변경하고 CSS로는 웹의 스타일을 지정 JS로는 웹의 동작을 구현할 수 있다  


------
주요 명령어  
------
const  


document.write  


console.log  


수업필기
--
null : 빈 값이며 null자체로 값  

let : 선언 이름은 부여했으나 공간은 배정되지 않은 상태(초기 값이 들어가있지 않은 상태)  

자료형 : 자료를 저장하는 형태를 제약하는 것
